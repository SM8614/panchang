# Panchang — Daily Vedic Hindu Calendar

A fully self-contained web panchang with accurate Vedic astronomy calculations.

**Live site:** https://YOUR-USERNAME.github.io/panchang

## Features
- Tithi (lunar day) with exact end time
- Nakshatram with exact end time
- Dual tithi/nakshatra shown when two fall in one day
- Sunrise & Sunset (NOAA algorithm, accurate to ~1 min)
- Rahu Kalam, Gulika Kalam, Brahma Muhurta, Abhijit Muhurta
- Yoga, Karana, Masa, Vara
- 15+ Indian cities + 35+ US cities
- Detect location via browser GPS
- Lahiri Ayanamsa · Meeus astronomical algorithms
- No server, no API — works offline from a single HTML file
